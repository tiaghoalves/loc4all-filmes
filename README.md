# Loc4all Filmes
Sistema de uma locadora de filmes

### Usage

```bash
npm intall
npm start
```

Crie um arquivo .env no root do projeto com suas configurações, siga como exemplo o [.env.example](https://github.com/tiaghoalves/loc4all-filmes/blob/master/.env.example)

### Testes

```bash
npm test
```
